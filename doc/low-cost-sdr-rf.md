# 低成本 SDR 平台

目前主流的低成本 SDR 平台主要有 HackRF，bladeRF 和 USRP，以下为参数表：

|            |     HackRF     |      bladeRF      |     USRP      |
| :--------: | :------------: | :---------------: | :-----------: |
|  频率范围  | 30 MHz – 6 GHz | 300 MHz – 3.8 GHz | 50MHz – 6 GHz |
|  基带带宽  |     20 MHz     |      20 MHz       |    20 MHz     |
|    双工    |     半双工     |      全双工       |    全双工     |
| 垂直分辨率 |     8 bit      |      12 bit       |    12 bit     |
|  采样频率  |    20 Msps     |      40 Msps      |    40 Msps    |
|  数据接口  |     USB 2      |       USB 3       |     USB 3     |
|    FPGA    |       0        |    40 ~ 115 k     |  75 ~ 150 k   |
|  微控制器  |    LPC43XX     |    Cypress FX3    |  Cypress FX3  |
