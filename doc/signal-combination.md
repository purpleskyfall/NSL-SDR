# 信号合并

为了将仿真的多个基带信号合并传输，需要将每个基带信号分别上转换到其对应的中心频率。本文档以 GPS 的 L1 和 L2 信号为例，介绍信号合并过程。

## 上转换

对于信号调制，需要将基带 I/Q 分量与一个复数的正弦波（代表 L1 和 L2 的载波）相乘来上转换到射频。这个复数正弦波可以表示为 $exp(j*2*pi*f*t)$，其中 $f$ 是载波频率，$t$ 是时间向量，$j$ 是虚数单位。

对于 L1 和 L2，有：

```pseudocode
# 伪代码示例，用于信号调制
carrier_L1 = exp(j * 2 * pi * f_L1 * t)  # L1 的复数载波
carrier_L2 = exp(j * 2 * pi * f_L2 * t)  # L2 的复数载波

# 上转换到 L1 和 L2 的载波频率
signal_L1_modulated = (I_L1 + j * Q_L1) * carrier_L1
signal_L2_modulated = (I_L2 + j * Q_L2) * carrier_L2
```

这里 `I_L1`，`Q_L1`，`I_L2` 和 `Q_L2` 是分别对应于 L1 和 L2 信号的同相和正交分量。`f_L1` 和 `f_L2` 是 L1 和 L2 信号的载波频率。

## 信号组合

一旦两个信号调制完成，便可以将它们相加来进行组合：

```pseudocode
# 伪代码示例，用于信号组合
combined_signal = signal_L1_modulated + signal_L2_modulated
```

这个组合信号 `combined_signal` 将包含两个不同的 GPS 信号，每个都在其对应的频率上。这个组合信号可以直接用于数字仿真或者经过数模转换后用于实际的信号发射。

以上介绍的技术称为频分复用（Frequency Division Multiplexing, FDM）。

## CEMIC

CEMIC（Constant Envelope Multiplexing via Intermodulation Construction）是一种用于复用多个信号的技术，它的主要目标是保持复用信号的包络恒定。即使在多个信号叠加的情况下，恒定包络可以减少信号通过非线性系统（如功率放大器）时的失真，从而提高功率效率并简化发射机的设计。

在传统的频分复用（FDM）或时分复用（TDM）技术中，复用的信号可能会在时间或频率上产生不同的峰值，导致复用信号的包络变化。这种包络变化在通过功率放大器时可能会引起信号的非线性失真，尤其是当放大器工作在接近饱和状态以提高效率时。此外，这种包络变化还会增加放大器设计的复杂性，因为它需要处理信号的动态范围。

CEMIC技术通过精心设计信号的相位和幅度，通过相互作用产生的内模（intermodulation）产品来确保即使在多个信号叠加的情况下，总的复合信号也具有恒定的包络。

采用 CEMIC（Constant Envelope Multiplexing via Intermodulation Construction）技术实现北斗 B1I 和 B1C 信号复用的方法涉及将这两种信号以一种特殊的方式结合起来，以确保组合信号具有恒定的包络。这样做的目的是减少在功率放大器和其他非线性系统中穿过时的失真，提高发射机的功率效率，并在接收端提供更好的接收性能。下面是一个简化的说明，介绍如何使用 CEMIC 技术复用 B1I 和 B1C 信号：

1. **信号选择**：
   - **B1I 信号**：通常采用二进制相移键控（BPSK）调制。
   - **B1C 信号**：采用更复杂的调制技术，比如二元偏移载波（BOC）调制或多重二进制偏移载波（MBOC）调制。
2. **调制信号的构造**：
   - 对于 B1I 和 B1C 信号，首先选择适当的基带信号。这些信号会有不同的数据和导航消息编码。
   - 对于 CEMIC 技术，要特别注意信号的相位和幅度，因为这两个参数将决定信号组合的方式和结果。
3. **相位和幅度的优化**：
   - 使用特殊的算法对 B1I 和 B1C 信号的相位和幅度进行优化，以保证在它们叠加时，总的信号包络保持恒定。
   - 这种优化通常是通过迭代算法完成的，可能涉及到求解非线性方程组。
4. **信号复用**：
   - 在相位和幅度被适当调整后，B1I 和 B1C 信号在基带进行线性叠加，然后通过一个共同的射频链路传输。
   - CEMIC 技术确保了即使在叠加的信号中存在内部调制互相作用，复合信号的包络也保持恒定。
5. **传输和接收**：
   - 由于复合信号的包络恒定，可以使用效率更高的功率放大器，因为它们可以在接近饱和点的状态下工作，而不会产生过多的失真。
   - 在接收端，接收机能够分离并分别解调 B1I 和 B1C 信号，即使它们是通过相同的频率和相位复用的。

通过 CEMIC 技术，B1I 和 B1C 信号可以共享同一个频段和发射天线，同时提高了系统的频谱效率和功率效率。
