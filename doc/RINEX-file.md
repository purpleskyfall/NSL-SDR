# RINEX 文件

本项目仅使用 RINEX 格式的导航文件，因此本文档仅介绍与导航文件有关的部分。

## 版本号

目前主流使用的 RINEX 版本主要为 RINEX 2.X 和 RINEX 3.X。RINEX 4.00 版本已发布，预计不久的将来会推广使用。

就导航文件而言，RINEX 2/3/4 三个版本格式是不兼容的。相比 RINEX 2，RINEX 3 版本的导航文件支持多系统星座；为了适应现代化的导航电文信息，RINEX 4 做了进一步的改进。

## 文件名

RINEX 2 格式的导航电文仅对 GPS 和 GLONASS 的广播星历做了标准定义，并且一个文件不能包含两种或以上系统的导航电文。广泛使用的 RINEX 2 格式的导航电文是 IGS 公布的广播星历文件，文件名为：

```html
brdc<doy>0.<yr>n
```

这里，`<doy>` 表示年积日，`<yr>` 表示 2 位数字表示的年份。

若需要一个包含所有导航卫星系统广播星历的文件，则可以下载 IGS 公布的 RINEX 3 格式的广播星历，文件名为：

```html
brdc<doy>0.<yr>p
```

或：

```html
BRDC00IGS_R_<year><doy>0000_01D_MN.rnx
```

这里，`<year>` 表示 4 位数字表示的年份。

## 实现细节

一般情况下，在一天内，GPS 卫星所有的播发相同的电离层参数；而 BDS 卫星播发的电离层参数是逐小时变化的，而且每颗卫星的电离层参数也不同。IGS 推荐优先使用 GEO 卫星的电离层参数，其次为 IGSO，最后为 MEO（RINEX 3.05）。

不过，总体上其改正效果差别不大。为了使程序逻辑不至于过于复杂，参考 RTKLIB，本程序仅使用最后一组电离层参数。

理论上，程序应实现 RINEX 4 格式兼容，但考虑到目前还没有主流机构公布测试数据，现阶段仅兼容 RINEX 2/3。
